Lists Usage
Unordered List -   An unordered list is a collection of values without any particular regard for ordering, e.g, ingredients list. Defaults to bullets. To start an unordered list, use the tag <ul>
Ordered List -   An ordered list is a collection of values in which the order of the values is specific, e.g., steps of a recipe. Defaults to numbers. To start an ordered list, use the tag <ol>
Definition List -   A definition list is a collection of values organized in pairs - the term and its definition. To start a definition list, use the tag <dl>
