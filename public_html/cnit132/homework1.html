The Turn of the Century

Introduction

I graduated Phi Beta Kappa from the University of California at Berkeley in December 2016. I currently work full time for the USDA and am pursuing a number of certificatons from the City College of San Francisco's Computer Science and Computer Networking & Information Technology Departments.

I enjoy learning HTML because its output is readily visible and practical. I have taken three Python classes at CCSF and took JavaScript simultaneously with this course. While I like learning the back-end side, I think HTML and JavaScript are more fun because you get to more clearly see the output. I only wish I had a better eye for design so my pages looked better.

One site that I find helpful when writing code is W3Schools. This site is useful because it provides tutorials for many of the most commonly learned languages in website and web application development. It also has specific articles that cover most of the common use cases with interactive code that allows you to see both how the code is structured and what the output looks like.

Interesting Working Titles

  • Web Content Accessibility Guidelines (WCAG) Overview - This site provides an introduction to making websites accessible to people with disabilities. This is both more inclusive and standard practice.
  • W3C Markup Validator - This is a free site that allows you to easily review your HTML is in compliance with the international agreed upon standards. Ensuring your site meets these will help it to be more successful.
  • Mobile-Friendlty Test - This free site allows you to enter either your websites code or its URL to test its ability to work well with mobile browers. This is critical as much of the content on the internet is through the use of a mobile device such as a smartphone these days.
